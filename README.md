## Tools that allows you to run home lab services!

#### Assets:
* Docker (container)
* ha2busybar (push home assistant data to Busy Bar)

#### Crontabs:
* Bedrock (Start Minecraft Bedrock at Boot)
* Java (Start Minecraft Java at Boot)
* Uptime (Start Uptime Kuma at Boot)
* ErsatzTV (Start ErsatzTV at Boot)

#### Development:
* Flipper Build All Apps (compile all binaries for user submitted Flipper Zero apps)
* Forgejo (Git)
* Hoppscotch (API)

#### Emulators:
* SWOTGB (Gameboy)
* Javatari (Atari 2600)
* Retroarch (General)
* WebЯcade (General)

#### Games:
* PlainChess (Chess)
* Minecraft (Classic)

#### Game Servers:
* Java (Minecraft Java)

#### Lifestyle:
* Mealie (Food)
* FreshRSS (News Feed)

#### Smart Home:
* Arduino/Soil Moisture (Soil Moisture Detecter)

#### Utilities:
* Draw.IO (Diagramming)
* Uptime Kuma (Server Monitoring)

###### Please Note:
* These projects are designed to work with Ubuntu Linux. No other operating systems are supported at this time.
* If you plan on hosting more than one, I recommend checking out proxmox: https://www.proxmox.com
